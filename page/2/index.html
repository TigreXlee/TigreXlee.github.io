我的三分钟热度

Posted on 2016-11-04

我不是个喜新厌旧的人，我喜欢的东西会一直喜欢，但是每个阶段侧重点不同。但是无论喜欢哪一种，都没有太深的钻研，浅尝辄止，热度三分钟。喜欢的东西包括但不限于以下几种

###音乐###
曾经有一段时间对音乐很着迷，硬件上买过不少随身播放器和一些很贵的耳机，买过CD机、艾利和播放器、iPod classic等播放器，还有创新Air、TF10、AKG450等耳机。然后就是去各大音乐论坛（PT80、杂碎、CD包）找各种的无损音源。热度过了，现在还是爱听音乐，但大部分时间都是直接用手机听了，主要是手机太方便了，再加上各种在线播放，几乎想听什么歌都有了。

###书法###
同样的，硬件方便也买了不少，草帽催的毛笔两只，后来搬家的时候丢了。各种钢笔，LAMY 狩猎者、Pilot 88G、英雄100、永生101、白金3770等，以及各种字帖，基本都是古帖，其中最爱的是虞世南《孔子庙堂碑》，放在床头也好久没翻了。另外就是《灵飞经》，适合练硬笔。还有就是看了很多教学视频，受影响最大的应该是田蕴章的《每日一题、每日一字》。现在也很少写了，偶尔签个名手都不稳了。

###EDC###
曾经一段时间也很着迷，但入坑不深，也就买了个莱泽曼多功能钳子（带到机场被迫丢掉了）、买了两个手电、两把刀、一个弹弓。那段时间总在栖兰小筑和生存着贴吧混。现在还是喜欢刀，蜘蛛的小折刀还是会随身带着，只是没有经历去玩了。

###ID###

我还是一个ID控，网上的ID一变再变，主要原因是喜欢的ID自己抢不到。偶尔脑子里出现一个词就会马上查一下邮箱、域名是否可以注册，这几乎成了一种病。注册过的邮箱不计其数了都。感觉最难的是Gmail，几乎能想到的词都被注册了，十亿的活跃用户真的不是盖的。目前这个ID应该会用下去，因为注册到了Gmail啊！

奇葩的一天

Posted on 2016-11-01

两个面试

上周一，一个广州的座机锲而不舍的打我电话，因为我当时在开会，所以没有接，连着打了六七次。第二天又打，我接了，是一家公司的面试邀请。我并没有投过这一家，是他们在网上直接搜到我的简历的。电话中跟他们约定好了面试时间，本周一。

周一一早，上班高峰期，我挤上了能把孕妇挤流产的地铁5号线，然后转乘能把女人挤怀孕的地铁3号线，到达终点站机场南的前一站下车，再换乘707路公交车晃晃悠悠来到了面试地点，鸟都不拉屎的地方。没想到的是，在门卫处HR告知研发部在工厂，即并没有在本部！所以又不得不坐上他们公司的车，这次总算是到达了真正的目的地，简直偏到姥姥家了。

奇葩的还不是这个，面试官是他们公司的王总,第一句话就是“你的简历并不太符合我们的职位要求” What ? (黑人问号脸)，“其他岗位倒是可以试试，但待遇和你要求的差的比较多” Fuck!我差点飙出脏话，我并没有投你们公司的简历，是你们的HR找的我，而且你们招人之前不先把简历看一遍再确定是否要邀请别人参加面试的么？浪费我请一天假，坐几个小时车到你们公司来面试，简直日了狗了！不过出于个人修养这些心理话并没有说出口。只是说了一句“那就算了吧，没有再谈的必要了”，然后转身离开了。

出来之后发现附近就一趟公交车，且不到地铁站，只得滴滴叫了一辆车。结果奇葩的事情发生了，我告诉了司机我所在的位置，就在地图上看到司机在我的周围一圈一圈的绕，却怎么也找不到我，绕了三圈的时候我和司机说实在找不到就取消吧，司机不愿意，说我都绕半天了，取消不是白绕了？又绕了三圈还是没找到，司机就主动放弃了。我只好步行一段路找到另一个公交站。

面试的时候有一个未接来电，就在我步行找公交的时候电话又打了过来。是我曾经面试过的一家公司，在今年的六月底，我正在办离职手续的时候去他们公司面试过，当时基本上都可以去上班了。结果第二天他们公司据说重新规划，把这个职位取消了。现在又打过来让我过去聊一下，正好趁请假这一天就打算过去看看。

下午到他们公司，简单的又聊了一下，就出来了，结果应该并不乐观。

第二天要回来上班，所以晚上就要过来东莞了。出来的时候刚好六点，因为刚好有BRT直达东站，且距离比较近，我就用支付宝买了7：05开的高铁票。心想只要上了公交车半个小时肯定能到，时间还是充裕的。不曾想，上的那趟车是短线，半路就都下车了。我一看距离也不远了，就又滴滴打了一辆车，结果司机打电话过来说路况不好，过来要十分钟以上！等不了，刚好打车时地图上看附近有地铁站，感觉跑去坐地铁。虽然换成两次，但还好在6：45赶到了东站。进站取票，顺便在汉堡王买个汉堡，单点了一个汉堡竟然40元。

顺利上车，在车上把汉堡干掉差不多就到了常平，价格虽高味道还是不错的。出来后又叫滴滴，上午的奇迹又出现了，司机师傅又再我附近绕圈圈！还好火车站附近标志性建筑多，绕了几圈之后终于接到了。

联想到周日的时候去佛山，按照导航到达目的地却怎么也找不到目标建筑，问了下人又打了个摩的才到达真正的目的地。这一切都是由于我使用的是Google map，在我大天朝是有偏移的，以前好像没这么严重，这几天格外厉害，看来要下载个高德了。

原来是你在作怪

Posted on 2016-10-27

从上周开始，手机流量掉的飞快。周六晚上，睡前手机流量还有近500M，一觉醒来竟然超用了300+M!扣了几十块钱，关键是根本就没开什么东西啊。赶紧买了1G的流量包，用了不到三天，又用完了！而且在宿舍和办公室都是用wifi，办公室的wifi是自己的360随身wifi，时好时坏，但是绝大部分时间是连着的，只有在中午吃饭和用微信的时候使用手机流量。尽管如此，1G的流量包也用不了三天。查看了所有app的流量使用情况，与实际掉的流量差数倍，真是见了鬼，为了防止流量偷跑，我安装了LBS手机监控，把所有app涉及上传、更新的都设置成wifi-only，使用绿色守护关掉除了ss外的所有后台程序，甚至重新刷了系统，清空了手机，所有app都到Google play重新下载安装，然后回到宿舍的时候关掉数据流量，避免晚上流量偷跑。即使如此流量都是掉的飞快，是我百思不得其解。

自从月初把Blog从Vultr搬到Bandwagon,一直用得都挺顺手的。VPS上面只搭建了Blog和SS，内存使用一直稳定在130M左右，512M的内存用起来绰绰有余了。然而前天打开后台看了看，内存竟然飙到了300多M，虚拟内存更是几乎爆掉了，当时并没有太在意，重启了一下服务器就降回去了。昨天又打开后台，发现内存又飙回去了，这次才开始重视，搜了一下查看进程的命令，用top命令看了下，其中PID 1034这个进程就占用了27%的内存，但是我又无法确定这是哪个进程，没敢关掉。当时快下班了，而且也没有影响到Blog正常使用，就先放下了。

今天又用top命令结合netstat -antup终于知道了PID 1034对应的进程和端口，找到了所有问题的症结，原来这一切都是kcptun在作怪！

上周我在VPS是部署了kcptun来配合SS，速度提高了很多，当时并没有仔细阅读相关文档，现在才知道它和net-speed一样是用带宽换延迟，而且是双边加速，根据参数的设定多消耗流量达4倍以上。难怪我的手机流量掉的那么快，我关掉了一切，只留下ss，然而流量就是从ss跑的，What the fuck!

kcptun以后只能在wifi的时候才打开了，珍惜流量，远离kcptun!

关于云盘

Posted on 2016-10-21

昨天，有人在微信群里说“360云盘关闭了，我的几十T的资料在明年2-1前根本下不完！”这是继华为云盘、新浪微盘、金山快盘等之后的又一家云存储关闭。其实我个人对这些并不太感兴趣，因为我是一个云盘轻度用户，我不太清楚那些动辄几十T资料的人会不会看他自己存的这些东西，如果看，这辈子能否看得完。像电视剧或电影这些我基本都会只看一遍，看完就删掉了。我对这些云盘的应用大部分是在下载某些资源的时候（大多是无损音乐），这些存在云盘上的资源一般都需要先转到自己账号下才能下载，所以我注册帐号多是为了这些。这也导致注册了之后就忘记了，下次用得到的时候再注册一次，从而大多数网盘的账号我都不确定。

我自己最常用的网盘一共有三个，一个是Dropbox，注册的最早，容量也比较小，还需要翻墙。不过用着比较放心，里面主要存了一些个人资料文档和证件照之类的。

第二个是Google Drive, 也需要翻墙，免费版 15G容量，主要用来同步手机的照片和备份手机上的部分资料。

第三个是国内的坚果云，免费版每月只有1G上传和3G下载的流量，用途和Dropbox差不多，主要是文档资料、证件照片和Blog备份等等。

趁着这次机会，打算整理一下自己的其他大容量网盘，百度云账号两个，上面主要是一些无损音乐，正常；360由于太长时间没有使用，账户被冻结，/汗。115垃圾，要登录必须下载他们的浏览器，忍住恶心下载了浏览器打开，结果密码忘记了。使用找回密码功能，到邮箱点开他们的连接，居然是403 Forbiden,What the fuck!!

近况

Posted on 2016-10-20

这大概是最尴尬的半年。

今年六月底，我从上家公司离职，开始边参加PMP培训边找工作。几乎同时，老婆也从她工作了八年多的公司离职，开始“创业”，说是创业，其实还是给别人打工。是和她的一个老乡合作，那个老乡是她哥的同学，在深圳开了一家财税咨询公司，因为打算在新三板上市，所有打算在广州也开一家分公司。于是就找到了我老婆，费用他们出，我们出力，会给一部分的股份，同时每月会给一点生活费。

经过多次的选址、注册，公司总算是办下来了。但是整个运作过程困难重重，首先没有资金支持，连公司开户的钱都要自己先垫付，这到还不是最严重的，更重要的是既不提供技术支持，也不提供人员支持。因为老婆也是初入此行，肯定有很多不懂的地方，但每次有问题要问深圳的时候，得到的答复只有一个“去问同行”，我其实很纳闷，你们不就是同行吗？另外就是公司成立之后就是一空壳，既没有做推广，也没有招业务员，就这样成功可想而知。终于在前几天，深圳的老板通过大舅子告诉我老婆，说开业几个月没啥绩效，打算把分公司关掉！白忙活了。

再说我这边，8-1进了现在这家公司。来了之后发现这是一家很奇怪的公司，首先，一般公司都会有的新员工入职培训完全没有，包括上下班打卡时间、公司部门人员介绍、发工资的时间、公司制度等等都完全没有讲。当然这并不是最主要的，主要的是来了这么久，完全没有人给安排工作！一个月的时候我问过部门经理，没有什么答复，后来又安排了一个同事跟着学工作相关事宜，现在除了每周开个电话会议，也没有什么事情做。现在试用期三个月快结束了，我完全不知道在几乎什么都没干的情况下怎么通过试用期。当然，通不过也没有什么可惜的，只是老婆现在的情况如果我再重新找工作会很紧迫。再加上又刚买了房子，房贷要开始还了～

当然，上班没有事情做也不是完全没有好处，好处就是在这段时间里我把PMBOK完整的看了一遍，并把五套模拟题做了两遍，最后顺利的通过了PMP考试，获得了PMP证书。也算是一点安慰吧。

Net-Speeder安装

Posted on 2016-10-18

A program to speed up single thread download upon long delay and unstable network
安装后速度加快，流量翻倍。

###环境：

VPS；Bandwagon

系统：Ubuntu 14.04 64bit

###步骤

1、下载源码
wget https://github.com/snooda/net-speeder/archive/master.zip

2、解压
unzip master.zip

3、准备编译环境

a、安装libnet-dev
apt-get install libnet1-dev

b、安装libpcap-dev
apt-get libpcap0.8-dev

4、编译(进入到net-speeder解压缩目录)
cd /net-speeder-master

sh build.sh -DCOOKED

5、把生成的net_speer复制到/usr/bin目录下
cp net_speeder /usr/bin

6、启动
/usr/bin/net_speeder venet0 "ip"

ps:此规则为加速所以ip协议数据

7、加入开机启动

echo ‘nohup /usr/bin/net_speeder venet0 “ip” >/dev/null 2>&1 &’ >> /etc/rc.local

—EOF—

《白夜行》的几点疑惑

Posted on 2016-10-14

1、桐原亮司杀父亲桐原洋介是唐泽雪穗策划的吗？
有可能，但没敢往深处想，如果是，那也太黑暗了。但又觉得应该不是，因为这里面有太多的惊险和巧合。
桐原亮司一定会杀父亲吗？他们父子关系一直很好，父亲很溺爱这个独子，也没有矛盾。
即使她相信桐原亮司会为了她杀掉父亲，一个11岁的孩子有把握杀掉一个成年男子吗？
我更倾向于她有意让亮司知道他父亲的变态行为，但杀父亲应该是意料之外。

2、唐泽雪穗的生母是自杀还是被杀？
肯定不是意外死亡，根据书中的描述，雪穗明显是带着钥匙的，她让房东的儿子去帮忙开门明显是获得不在场证明。伪造意外的死亡的现场也肯定是雪穗做的或者参与的。但她母亲是自己喝下的酒和药，还是雪穗谋杀了自己的母亲始终没有搞清楚。因为没有发现她母亲要自杀的直接动机，但骗使母亲喝下过量药物的可行性？存疑。

3、桐原亮司因为奸尸留意下的阴影导致不射？
这是网上一些人的推测，但个人并不同意此说法。因为在此之前，亮司做皮条客那一段就有对他不射做过侧面描述，

“有本事就让它射啊！”他双手夹住奈美江的脸，把阳具顶到她面前，“用手用嘴随便你，要用下面也可以。你以为我撑不了多久，是不是？那你就试试看！”

另外根据唐泽雪穗事发当晚外出，以及死者房间要洗发露，可以知道雪穗当时去了酒店帮桐原亮司。所有是雪穗帮桐原亮司射出来然后装入了死者体内。

4、唐泽雪穗只有面对桐原亮司下体才能湿？
这也是网上一些人的看法，实际并非如此。唐泽雪穗应该是可以控制自己的欲望，而且根据书中的描述，她和高宫诚并不是一直如此。

诚不认为自己的做法有问题，因为不久之前，这样便足以产生充分的润滑。

5、唐泽雪穗和桐原亮司之间是爱情吗？
我觉得是爱情，无论从灵与肉。桐原亮司是唐泽雪穗心目中的太阳，

我的天空里没有太阳，总是黑夜，但并不暗，因为有东西代替了太阳。虽然没有太阳那么明亮，但对我来说已经足够。凭借着这份光，我便能把黑夜当成白天。你明白吧？我从来就没有太阳，所以不怕失去。

6、桐原亮司的死是唐泽雪穗策划的吗？
有人说，最后是唐泽雪穗设计杀死的桐原亮司，从而成功上岸。
肯定不是，他们是枪虾与虾虎鱼的关系，有着深深的羁绊。从桐原亮司死时唐泽雪穗超乎常人的表现就能看出她的悲痛。

笹垣脚步蹒跚地走出警察们的圈子。只见雪穗正沿扶梯上楼，她的背影犹如白色的影子。 她一次都没有回头。

枪虾和虾虎鱼

Posted on 2016-10-14

历时半个多月，终于时断时续的看完了《白夜行》。

其实从看完《嫌疑人X的献身》，我就喜欢上了这个叫东野圭吾的作家，这个看完《白夜行》，感觉更加强烈。

首先不得不说，东哥的叙事能力太强了，整个故事历时二十年，中间穿插各个分枝，每一章看似各自独立却又处处皆是伏笔。整本书看下来就好像玩拼图，拿到的每一个碎片都是一个故事，但是故事中必定有一点和其他故事相关联，直到拼到最后一块，所有的疑团才全部解开，才发现每一块都独立却又不可或缺。果然是大师的文字没有一句是废话。

枪虾和虾虎鱼是故事中的警察笹垣来形容男女主角桐原亮司和唐泽雪穗之间的共生关系的。

枪虾通常是盲的，一个枪虾会挖一个洞，然后一对虾虎鱼会游来同住，作为挖洞的回报，枪虾会受到虾虎鱼的保护，甚至得到食物。通常虾虎鱼坐在洞穴的入口处，枪虾忙着清理通道，当枪虾出来倾倒沙石时，它总把一根触须搭在虾虎鱼的身上，其他鱼来袭时，虾虎鱼一动身，枪虾就逃回洞中。

当然，他们的所作所为是被人不齿的，也是违法犯罪的。但抛开这个层面，又不得不佩服男女主角的心思缜密和聪明早慧。十一岁的小孩子便可有计划有预谋的杀人，并瞒过了所有人，不得不令人惊叹和震惊。
后来的一切几乎都在他们的掌控之中，桐原亮司为唐泽雪穗扫清一个又一个障碍，从未失手。唐泽雪穗也在不知不觉中帮助同源完成财富的原始积累。如果不出意外，那么他们手牵手走在阳光下或许不是梦。

我的天空里没有太阳，总是黑夜，但并不暗，因为有东西代替了太阳。虽然没有太阳那么明亮，但对我来说已经足够。凭借着这份光，我便能把黑夜当成白天。我从来就没有太阳，所以不怕失去。–唐泽雪穗

雪穗从不以真面目示人，亮司则至今仍在黑暗的通风管中徘徊。
