New Moto X Andriod 6.0 update

Posted on 2016-10-24

最近突然想升级一下我的Moto X+1,因为解锁以后官方推送到OTA是不能升级的，一升就变砖了，只能自己下载底包刷机。这次刷的是Andriod 6.0。

刷机步骤如下:

准备：

Moto X2专用adb及fastboot组件 密码: ctrf

Andriod 6.0底包 密码: m48c

1、Andriod 6.0 底包下载完后解压缩，提取出gpt.bin、boot.img、system.img_sparsechunk.0~9
放到刷机组件文件夹里。

2、刷入bin文件
fastboot flash partition gpt.bin

3、刷入boot.img
fastboot flash boot boot.img

4、刷入system.img_sparsechunk.0~9
fastboot flash system system.img_sparsechunk.0

fastboot flash system system.img_sparsechunk.1

``````````` fastboot flash system system.img_sparsechunk.9

5、wipe (所有文件消失)

fastboot -w

6、重启

fastboot reboot
